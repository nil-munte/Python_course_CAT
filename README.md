# Python_course_CAT

**Introduction to Python Programming** (in Catalan)

## Description

This repository provides an introductory Python programming course designed for Catalan speakers. It includes Jupyter Notebooks and supporting materials that guide students from basic programming concepts to the fundamental building blocks of the Python language.

## Course Objectives

- Introduce programming concepts from scratch in a clear and progressive way.  
- Teach Python fundamentals: syntax, data types, operators, data structures, control flow, and functions.  
- Offer practical examples and explanations tailored to beginners.  
- Encourage hands-on learning through interactive notebooks and experimentation.

## How to Use the Materials

1. Install Python and a suitable environment (e.g., Anaconda or Python + pip).  
2. Open the `.ipynb` notebooks using Jupyter Notebook, JupyterLab, or VS Code.  
3. Follow the lessons in order to build a strong foundation step by step.  
4. Modify and run the code examples to deepen your understanding.  
5. Refer to the installation guide PDF if you need help setting up your environment.
